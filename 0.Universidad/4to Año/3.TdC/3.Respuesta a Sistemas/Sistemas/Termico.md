# Introducción 
Resistencia térmica:
	$R=\Delta T / q$     ; $q = C.dT/dt =$  transferencia de calor
	- Inducción: $R=$
	- Convección: $R=$
	- Radiación: $R=$
Capacidad térmica:
	$C=m.c=\frac{\Delta q}{dT/dt}$ 
	- $m$ -> masa
	- $c$ -> capacidad calorifica (depende del elemento)
Tasa de intercambio:
	$T_i= \Delta q = m.c.dT/dt$
