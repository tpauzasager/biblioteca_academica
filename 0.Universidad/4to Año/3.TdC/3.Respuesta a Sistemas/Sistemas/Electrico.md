# Introducción 
Inductor
	- Voltaje: $v=L.\frac{di}{dt}$	
	- Corriente: $i=1/L.\int{v.dt}$
	- Energía almacenada: $E=L.i^2/2$
Capacitor
	- Voltaje: $v=1/C.\int{i.dt}$
	- Corriente: $i=C.\frac{dv}{dt}$
	- Energía almacenada: $E=C.v^2/2$
Resistencia
	- Voltaje: $v=R.i$
	- Corriente: $i=v.R$
	- Energía disipada: $P=v^2/R$
