# IGMP
Protocolo de Administración de Grupo en Intenet

Se usa para gestionar la recepción de tráfico multicast (1 Rx, muchos Tx) de una red IP

Definición:
	- De multidifusión (intercambio de info entre routers)
	- Transmite datagrama IP a un conjunto de dispositivos
	- Se propaga por una o varias redes físicas
