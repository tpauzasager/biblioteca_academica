**Control de Enlace Lógico**
# Definición 
- Capa 2.1 
- Transmisión de PUD entre dos estaciones (sin necesidad de comnutación)
# Caracteriticas
- Admite **acceso múltiple** 
- Capa MAC lo descarga de detalles del acceso al enlace (?)
# Servicios 

|     | Orientado a conexión | Con confirmación | Descripción | Control de      | Garantiza          |
| --- | -------------------- | ---------------- | ----------- | --------------- | ------------------ |
| 1   | No                   | No               | Datagrama   | -               | -                  |
| 2   | Si                   | -                |             | Flujo y errores | Recepción de datos |
| 3   | No                   | Si               |             |                 |                    |
Conexión:
	- Antes de comenzar transferencia, se establece conexión lógica entre Tx y Rx
	- Permite identificar conexión y tener control de flujo 
	Ventaja -> **Control de flujo**
	Denventaja -> Retardo inicial
Confirmación:
	- Luego de cada transmisión de trama, Rx envía confirmación de recepción 
	- Si no recibe confirmación => Retransmisión de trama 
	Ventaja -> **Control de errores** 
	Desventaja -> Lentitud
