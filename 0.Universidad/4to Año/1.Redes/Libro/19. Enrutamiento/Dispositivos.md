# DHCP
Protocolo de Configuración Dinámica de Host 

Función 
	- Asignación dinámica de dir IP para dispositivos

Protocolo de tipo Cliente-Servidor (el servidor es el que otorga la dir IP al cliente)
___
# DNS
Sistema de Nombre de Dominio 
Permite traducción de dominia a dirección IP
Guarda en BD relación entre dir IP y Dominio
	Si no tiene la relación => Va saltando entre servidores hasta conseguirlo
	Son gerárquicos => Los saltos son hacia servidores de mayor gerarquia
Tiene puertos UDP y TCP
