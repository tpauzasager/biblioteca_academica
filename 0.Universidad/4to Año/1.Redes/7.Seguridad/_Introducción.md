Conceptos:
	Confidencialidad (privacidad) -> Que no se pueda acceder 
	Autenticidad -> Datos provienen del origen indicado 
	Integirdad -> No se modifican los datos 

Ataques:
	Interceptación
	Fabricación
	Modificación
	Destrucción

Métodos
- Clave de acceso 
- Encriptado 
- Seguridad física
- Firma digital 
	- Criptografia asimétrica (Clave privada y pública)
	- Requiere autoridad certificante (para distribuir clave publica)
	- Provee autenticidad, integridad y no repudio
- Firewall 
	- Divide red interna de la externa 
	- Control de accesos
	- Protección de ataques externos 
	- Limita tráfico de servicios vulnerables 
- Capacitación usuarios
- IPSec 
	- Conjunto de protocolos de seguridad (para encriptado y autenticidad)
	- Modos: 
		- Transporte -> No asegura capa IP; Va de extremo a extremo
		- Tunel -> Asegura incluyendo capa IP (3); Requiere Gateway adicional 
- VPN
