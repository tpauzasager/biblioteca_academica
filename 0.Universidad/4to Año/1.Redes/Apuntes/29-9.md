Tipos de Comutadores 
	de Circuitos
		- Hay monopolización de recursos (una sola comunicación por canal)
		- Enlace físico
		- Conexión física
		- Transmisión continua
		- Eficiente para voz
		- Congestión al establecer conexión
	de Paquetes
		- Enlace lógico (con o sin conexión)
		- No hay monopolización de recursos
		- Información transmitida por paquetes aislados
		- Solo para digital
		- Eficiente para datos
		- Congestión en todo momento
		Tipos
			Circuito Virtual
				- Conexión virtual
				- TCP
				- Almacenamiento del paquete hasta el envío
			Datagrama
				- Sin conexión
				- IP, UDP
				- Almacenamiento opcional

Tamaño de Paquete
	Mayor tamaño => 
		- Mayor eficiencia (menor cant de headers)
		- Necesito menor BER (pq paquetes perdidos cuestan más)
	Menor tamaño =>
		- Menos eficiencia
		- Menos costo por perder un paquete

Conmutación de circuitos
	Fases
		1. Establecimiento del circuito 
		2. Tx
		3. Desconexión del circuito 
	Componentes
		- Abonado -> Terminal
		- Bucle local -> Conexion entre abonado y conmutador
		- Centrales -> Conmutadores 
		- Lineas principales/troncales -> Conexion entre conmutadores
	Tipos
		- División de espacio (analógicos)
		- TDM (en el tiempo)
	Hay efecto de concentración que afecta a cant de comunicaciones simultaneas

PPP (Poin to Poin Protocol)
	- Protocolo de 2do nivel (enlace)
	- Para comunicación serial (P2P)
	- Hay autenticación 
	- Asignación dinámica de IP
	- Se usa para formar vpn 
	PDU
		- Bandera (8)
		- Dirección (8)
		- Control (8)
		- Identificador protocolo (16)
		- FCS (CRC16 o 32)
		Campos dirección y control pueden no usarse
