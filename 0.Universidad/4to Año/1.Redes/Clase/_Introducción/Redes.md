# Redes 
**Definición** -> Conjunto de recursos que forman sistema de comunicaciones (transporte de información)
**Elementos**
	- Equipos determinados -> DTE
	- Nodos de red -> Permiten transporte de red
	- Enlaces de comunicación -> Vinculo entre DTEs y Nodos 
**Tipos**
	- P2P
	- Conmutada 
	- P a Multipunto
	- Internet 
**Tipos enlaces**
	- Dedicado -> No se comparte, sin intermediario entre Tx y Rx
	- Conmutado -> Se comparte el medio
		- De circuito -> Monopolización del canal (con conexión física)
		- De paquete -> Sin monopolio (se comparte en secuencias de paquetes)
**Señalización** -> <no_se_entiende>
# Clasificación Redes
**Area geográfica** 

|                        | LAN      | WAN      |
| ---------------------- | -------- | -------- |
| Distancia              | -        | +        |
| $V_t$                  | +        | -        |
| BER                    | -        | +        |
| Enlace                 | difusión | P2P      |
| Restricciones externas | no       | si       |
| Recursos               | privados | publicos |
**Ambito**
	- Publicas -> PSDN, PSTN
	- Privadas -> RPV (red privada virtual)
**Medio de operación**
	- Virtuales -> PVC, SVC (con conexión)
	- Datagramas -> sin conexión 
**Tecnología**
	- Analógica
	- Digital
