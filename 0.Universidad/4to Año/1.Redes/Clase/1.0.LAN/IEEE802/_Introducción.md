# Capas
## Fisica
- Codificación/Decodificación de señales 
- Generación/Eliminación de preámbulo (para sincronización)
- Transmisión/Recepción de bits
## Enlace de Datos
Control de Enlace Lógico (**LLC**)
	- Interfaz con capas superior
	- Control de errores y flujo 
Control de Acceso al Medio (**MAC**)
	- Ensamblado y desensamblado de datos en tramas con:
		- Direccionamiento (origen y destino)
		- Detección de errores
