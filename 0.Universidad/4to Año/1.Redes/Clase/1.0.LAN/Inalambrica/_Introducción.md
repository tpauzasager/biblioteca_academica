# Aplicaciones
- Ampliación de redes (Acess point)
- Interconexión de edificion (LAN to LAN)
- Acceso nómade (de computadora portátil)
# Requisitos
- Rendimiento 
- Cant nodos 
- Conexión a LAN troncal -> Conectarse a LAN cableada
- Área de servicio 
- Consumo de batería 
- Robustez (transmisión y seguridad)
- Funcionamiento de redes adyacentes y sin licencia
- Handoff (paso entre AP) / Roaming (handoffs que va haciendo para moverse libremente por area de cobertura)
- Config dinámica 
# Tecnologías 
**Infrarrojos** (IR)
- Haz dirigido 
- Omnidireccional 
- Difusión (use de reflector)
**Radio por espectro expandido** 
- Tecnicas -> Salto de frecuencia ; Secuencia directa
- Bandas: 900Mhz; 2,4Ghz; 5,8Ghz; 6Ghz
- Topología -> Concentrado o P2P
- No necesita licencia ENACOM
**Radio de banda estrecha** 
- Radioenlace
- Bandas: 18Ghz (con ENACOM); 5,8Ghz (sin ENACOM)
