# Técnica de Spread Spectrum
Por salto de frecuencia (FH - frecuency hooping)
**Prestaciones**:
	- Inmunidad al ruido 
	- Varios usuarios usando mismo ancho de banda (con poca interferencia)
	- Compensan pérdida de eficiencia espectral ($V_{Tx}/\Delta B$) 
	- Usado para CDM (multiplexación) y CDMA (acceso múltiple)
**Funcionamiento**:
	- Datos son modulados por FKS (frecuencia)
	- Se multiplica por un generador pseudoaleatorio (o pseudoruido)
**Como se produce**:
	- Cada simbolo se envía en frecuencias 2 o más frecuencias diferentes 
	- Mismo ancho de banda, pero se usan varias distribuciones de AB para mandar simbolos
