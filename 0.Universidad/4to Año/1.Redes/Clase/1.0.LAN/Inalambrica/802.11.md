# Subaca MAC 802.11
## Protocolo de entrega de datos 
2 Tramas -> equivalente a ARQ
	- Trama ACK (Time out)
	- Datos
4 Tramas -> Evita colisiones 
	- RTS (Request to send)
	- CTS (Clear to send)
	- Datos
	- ACK 
## Control de accesos
Opciones:
	DCF (Función de Coordinación Distribuida) 
		-> CSMA/CA (con contienda)
		-> Descentralizado
	PCF (Función de Coordinación Puntual) 
		-> Sin contienda
		-> Centralizado => No hay colisión
