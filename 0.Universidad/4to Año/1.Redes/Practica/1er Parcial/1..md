	c. La técnica de SS consiste en aumentar el AB soportado por el sistema empleando una modulación particular a la señal de datos. Tiene el objetivo de aumentar la seguridad
	Métodos
		FH: Salto de frecuencia. Consiste en respetar un AB fijo pero sufriendo un desplazamiento (en el eje de frecuencias) para cada paquete envíado. Este salto se realiza a partir de un modulador de pseudo ruido. Este método permita que multiples comunicaciones se lleven a cabo en simultaneo en el sistema, así como seguridad y resistencia a cierto tipo de ruidos
