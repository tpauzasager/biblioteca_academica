# Definición 
Conducta en interferencia intersubjetiva (conducta de cualquiera choca con la conducta de otro)
	Conjunto de normas de conducta obligatorias establecidas por el Estado 
___
# Clasificación 
## Público
- Orientado al interés público (del conjunto social)
- Relaciones jurídicas en donde Estado es parte
### Composición 
| Nombre                | Definición                                                         |
| --------------------- | ------------------------------------------------------------------ |
| Constitucional        | Determinan organización del Estado y poderes públicos              |
| Penal                 | Para castigar conductas ilícitas                                   |
| Administrativo        | Reglamenta organización y funcionamiento de admin pública          |
| Internacional Público | Relaciones entre estados soberanos                                 |
| Procesal              | Reglamenta org del **poder judicial** y competencias de tribunales |
## Privado
- Resguardan intereses privados 
- Relación jurídica de las personas entre sí 
### Composición 
| Nombre                | Definición (regula...)                                 |
| --------------------- | ------------------------------------------------------ |
| Civil                 | Relación entre personas                                |
| Comercial             | Actividad comercial entre personas (humano o jurídico) |
| Laboral               | Relación entre empleados y empleador                   |
| Internacional Privado | Derecho privado con sujetos extranjeros                |
| Minero                | Explotación de riquezas del subsuelo                   |
| Rural                 | Relaciones de vecindad rural                           |

## Natural
- Normas universales del hombre por ser humano
## Positivo
- Conjunto de normas jurídicas vigentes en un tiempo y lugar determinado 
## Interno
- Regula relaciones entre Estados y habitantes 
## Internacional 
- Regula relaciones entre países y estados soberanos
