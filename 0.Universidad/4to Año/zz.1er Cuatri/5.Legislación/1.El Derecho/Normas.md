# Definición 
Descripción de una conducta (forma de actuar) que debe ser cumplida 
# Para crearla
Se necesita:
	- Determinar cual conducta se debe seguir
	- " Porque se crea la norma
___
# Tipos
| Nombre    | Descripción                                                  |
| --------- | ------------------------------------------------------------ |
| Religiosa | Por personas que practican la fe                             |
| Moral     | Regulan accionar de la persona desde su conciencia           |
| Social    | Aquellos comportamientos que la sociedad considera adecuados |
| Jurídica  | Forma de actuar ordenada por el Estado                       |
## Jurídica
Denominadas **Leyes**
### Caracteristicas
| Caracteristica                       | Definición                                            |
| ------------------------------------ | ----------------------------------------------------- |
| Regulación de relación entre hombres |                                                       |
| Exterioridad                         | Solo toma en cuenta los actos (no los actos internos) |
| Imperatividad                        | Regla de conducta impuesta                            |
| Generalidad                          | Igual para todos                                      |
| Coercibilidad                        | Amenaza de sanción en caso de incumplimiento          |
| Coactividad                          | Facultad del Estado de exigir cumplimiento            |

### Valores Jurídicos
- Justicia
- Orden - Seguridad
- Paz - Poder
- Solidaridad - Cooperación 

### Relaciones Jurídicas
| Nombre  | Definición                     |
| ------- | ------------------------------ |
| Crédito | Derecho de exigir algo a otro  |
| Deuda   | Obligación jurídico de cumplir |
