
| Modo                      | Definición                                                                                                |
| ------------------------- | --------------------------------------------------------------------------------------------------------- |
| **Pago**                  | Deudor cumple obligación<br>Por consignación -> Acreedor no quiere cobrar (deudor paga a través del juez) |
| **Novación**              | Reemplazar obligación con una nueva                                                                       |
| **Trasacción**            | Obligación no cambia<br>Ambas partes ceden algo                                                           |
| **Compensación**          | Dos personas se deben mutuamente<br>Solo se paga la diferencia                                            |
| **Confusión**             | Acreedor == Deudor <br>Desaparece obligación                                                              |
| **Renuncia**              | Acreedor decide no cobrar                                                                                 |
| **Imposibilidad de pago** | Fuerza mayor, deudor no puede cumplir obligación                                                          |
| **Remisión de deuda**     | Acreedor entrega documente de dueda                                                                       |
| **Dación en pago**        | Deudor ofrece algo distinto a lo que debía                                                                |
