# Civiles
- Protegen habitantes y ciudadanos 
- Individuales:
	- Pertenecen a la persona individualmente
- Incidencia colectiva:
	- Pertenecen a todos los miembros de una comunidad (como integrantes, no de manera propia)

# Políticos
- Derechos republicanos de **participación política** (solo ejercido por ciudadanos)

# Sociales
- Protegen sectores más vulnerables de la sociedad
