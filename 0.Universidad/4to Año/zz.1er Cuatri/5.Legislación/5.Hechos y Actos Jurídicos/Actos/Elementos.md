# Sujeto
- Personas que actuaron con voluntad en el acto jurídico 
- Son **parte del acto jurídico** => Se llaman **Partes**
# Objeto
- Bien o hecho **sobre el que recae el interés** de las partes 
# Causa
- Origen del acto (**finalidad** por la que se celebra)
# Forma 
- **Modo de exteriorización** de voluntad de los sujetos 
- **Forma exitgida** para validez => Acto **formal** 
- **Forma no exigida** para validez => Acto **no formal**
## Escrita
- Instrumentos **públicos** 
	- Llevados a cabo según la ley 
	- Tienen **autenticidad**, **veracidad** y **fecha cierta**
- Instrumentos **privados** 
	- Sin interverción de Oficial Público 
	- No tienen lo que tienen el publico
