# Definición 
- Acto **voluntario** que **incuple la ley** y genera daño a otra persona
- Requisitos:
	- Voluntad (del que lo realiza)
	- Acto reprobado por ley 
	- Con **dolo** o **culpa**
	- Daño resultante 
# Tipos

| Delito   | Cuasidelito                                                                                                                                                                 |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Con dolo | **Imprudencia** -> Actuar sin precaución <br>**Impericia**      -> Falta de conocimiento técnico<br>**Negligencia**  -> Omisión de cuidad / No actuar cuando era debido<br> |
