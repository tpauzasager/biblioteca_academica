# Anormal
## Razones 

| Nombre                        | Definición                                                                                                                                                                                                                                                                                                   |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Imposibilidad de cumplimiento | Por **caso fortuito** o de fuerza mayor                                                                                                                                                                                                                                                                      |
| Rescisión                     | **Por acuerdo**<br>Bilateral:<br>   - Extingir relación contractual a futuro (no retroactiva). <br>   - Puede no afectar a todas las partes (si hay 3 o más partes) <br>Unilateral:<br>   - Solo una parte cancela contrato (sin tener en cuenta las demás)<br>   - Debe estár permitido por el contrato/ley |
| Revocación                    | - Acto **unilateral** para donaciones y mandatos<br>- Contrato sin efecto por causa jurídico o ley<br>- No retroactiva                                                                                                                                                                                       |
| Resolución                    | - Causado **por hecho posterior** a vigencia del contrato (previsto por contrato o ley)<br>- Retroactivo<br>- Busca **restaurar el patrimonio** de las partes a como estaba antes del contrato                                                                                                               |
| Imprevisión                   | - Cuando **prestación se encarece excesivamente** <br>- Se puede anular o ajustar contrato                                                                                                                                                                                                                   |
## Ejemplo 
### Revocación 
Un mandante otorga un mandato a otra persona para que actúe en su nombre. Puede revocar ese mandato en cualquier momento, dejando sin efecto las obligaciones futuras.
### Imprevisión 
Tengo un contrato por un servicio y se vuelve muy caro, puedo pedir cancelarlo o acordar con la otra parte para que mejore el precio.
