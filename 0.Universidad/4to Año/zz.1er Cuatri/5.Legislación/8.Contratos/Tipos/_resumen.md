| Contrato           | Caracteristicas                                  | Definición                                                                                                                         |
| ------------------ | ------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| Compra-Venta       |                                                  | Vendedor se obliga a transferir propiedad de la cosa (dominio) al comprador a cambio de prestación en dinero                       |
| Boleto CV          |                                                  | Adjudica prioridad sobre el derecho de 3ros que tengan medidas cuatelares sobre inmueble                                           |
| Permuta            | Real                                             |                                                                                                                                    |
| Locación           |                                                  | Locatario otoroga tenencia (uso y goce) temporal de la cosa, locador paga precio                                                   |
| Obra               |                                                  | Obligación de resultado                                                                                                            |
| Servicio           |                                                  | Obligación de medio                                                                                                                |
| Mandato            | Gratuito/Oneroso<br>Nominado                     | Mandatario se obliga a realizar actos en favor del mandante                                                                        |
| Comondato          | Unilateral<br>Gratuito<br>Real<br>Nominado       | Comodante entrega cosa n**o fungible** al comodatario como prestamo (debe ser restituido)                                          |
| Mutuo              | Gra/Oneroso                                      | Mutuante entrega al mutuario cosa fungible y este le devuelve en misma cantidad, calidad y especie                                 |
| Donación           | Unilateral<br>Gratuito<br>Nominado               | Donante transfiere dominio de cosa al donatario                                                                                    |
| Fianza             | Unilateral<br>Accesorio                          | **Fiandor** se obliga a cumplir obligación ajena en caso de incumplimiento<br>Fiador -> Garantía personal (con todo su patrimonio) |
| Depósito           | Uni/Bilateral<br>Gra/Oneroso<br>Real<br>Nominado | Entrega de cosa para guardado y posterior restitución                                                                              |
| Administrativo     | Nominado<br>                                     | Entre Estado y particular (con fines públicos)                                                                                     |
| Cesión de derechos | Uni/Bilateral<br>Gra/Oneroso                     | Transferencia voluntaria de un derecho del cedente al cesionario                                                                   |
