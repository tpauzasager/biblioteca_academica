# Definición 
- Mandatario se obliga a realizar uno/s actos jurídicos en nombre del mandante 
- Mandante puede o no conferir poder para ser representado (si no, mandante actua en nombre propio y en interés del mandante)
## Tipos
Expreso:
	- Contrato verbal o por instrumento público o privado 
Táciso:
	- A actúa en interés de persona B
	- B se entera y no hace nada para impedirlo 
Especial:
	- Un solo acto
General:
	- Un conjunto de actos 

## Con o sin representación
Con representación:
- El mandante responde directamente con el 3ro (mandante tiene responsabilidad)
Sin representación:
- 3ro no tiene conocimiento del mandante, se comunica directamente con el mandatario (este tiene la responsabilidad)
# Obligaciones

| Mandatario                                                                   | Mandante                                                               |
| ---------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| Cumplir actos comprendidos                                                   | Propocionar al mandatario medios necesarios para ejecución del mandato |
| Mantener informado al mandante por circunstancia de interes                  | Indemnizar pérdida sufrida por mandatario                              |
| No difundir información                                                      | Pago de retribución cuando corresponde                                 |
| Dar aviso al mandante de valor recibido en razón del mandato                 |                                                                        |
| Responsabilizarse por daño sufrido por ejecución (o inejecución) del mandato |                                                                        |
# Extinción 
Por:
	- Transcurso del plazo 
	- Cumplimiento de condición resolutoria
	- Ejecución del negocio 
	- Revocación del mandatario 
	- Muerte/Incapacidad del mandate/mandatario
___
# Ejemplos 
Sin representación:
	Pedro le pide a Lucía que compre una computadora para él, pero ella la compra a su nombre, con su dinero. Luego, Pedro le devuelve el dinero. El vendedor solo conoce a Lucía.
