# Definición 
- Estado para cumplir con fines puede actuar por:
	- Ejercer por si mismo (recaudar impuestos, proveer seguridad, etc)
	- **Contratar particulares que colaboren para realizar actos**
- Estado es una de las partes 
- Objeto: Satisfacción de finalidades públicas
- Se rigen por normal especiales creadas a ese efecto 
- Se formaliza por **licitación pública** (excepcionalmente por compra directa)
