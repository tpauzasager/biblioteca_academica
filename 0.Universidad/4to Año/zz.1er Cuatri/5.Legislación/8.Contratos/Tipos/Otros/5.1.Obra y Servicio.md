# Definición 
- Cuando persona contratista (o prestador de servicio) se obliga a favor de otra persona: 
	Comitente -> Realizar obra material
	Intelectual -> Proveer servicio mediante retribución 
- No hay dependencia laborar (diferencia con contrato de trabajo)
- Servicio -> Realizar actividad **independientemente de eficiencia**
- Obra -> Se promete **resultado eficaz**
