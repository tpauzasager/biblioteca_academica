# Definición 
- Contratista se compromete a realizar y entregar servicio 
- **Obligación de resultado**
# Elementos
## Objeto
- Ejecución o entrega de obra terminada (material o intelectual)

## Obligaciones

| Contratista                             | Comitente                   |
| --------------------------------------- | --------------------------- |
| Aportar trabajo y materiales            | Cooperación con contratista |
| Asesorar a comitente de su especialidad | Recepción de obra terminada |
| Respetar derecho del dueño              | Pagar                       |
| Entregar obra en tiempo y forma         |                             |
| Responsabilizarse                       |                             |
### Responsabilidad contra 3ros
- Empresario responsable ante 3ros frente a daños causados por obra

# Conclusión 
- Cumplimineto de obligaciónes de las partes
- Decisión unilateral del dueño 
- Fallecimiento o incapacidad de contratista
- Imposibilidad de cumplimiento
- Incumplimiento de comitente
