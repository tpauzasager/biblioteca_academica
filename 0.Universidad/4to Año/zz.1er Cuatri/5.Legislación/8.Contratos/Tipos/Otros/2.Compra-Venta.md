# Definición 
- **Vendedor** se obliga a transferir propiedad de una cosa al **comprador** 

# Requisitos
- **Consentimiento** de ambas partes (por transferencia de dominio y precio)
- La cosa (**enajenación permitida** por ley)
- **Precio en dinero** (determinado o determinable)
### Capacidad
- Vendedor -> capaz de disponer sus bienes
- Comprador -> capaz de obligarse
### Incapacidad
- Declarada por juez o ley
- Funcionario públicos no pueden sobre bienes administrados
- No sobre litigios en los que se internvino (jueces, funcionarios, auxiliares)
- Conyuges entre sí (bienes pertenecen a ambos)
# Carácteristicas 

| Nombre          | Definición                                                          |
| --------------- | ------------------------------------------------------------------- |
| **Oneros**      | Intercambio a cambio de dinero                                      |
| **Bilateral**   | Ambas partes obtienen ventaja (algo a cambio)                       |
| **Conmutativo** | Ventajas/Pérdidas se conocen antes de celebración                   |
| Formalidad      | Formal: Compraventa de inmueble<br>Informal: Combraventa de gaseosa |
# Obligaciones

| Comprador             | Vendedor                                                                                                 |
| --------------------- | -------------------------------------------------------------------------------------------------------- |
| Pagar precio acordado | Entregar cosa vendida <br>Realizar actos pertienentes (para transmisión de dominio)                      |
| Recibir cosa          | Conservar cosa vendida hasta su entrega                                                                  |
|                       | Recibir precio acordado                                                                                  |
|                       | Dar **garantí por vicios redhibitorios** (defectos ocultos) y **evicción** (obligaciones de saneamiento) |

# Boleto de Compra-Venta
## Definición 
- **Pre-contrato** usado para **inmuebles**
- **No se transfiere** la propiedad hasta que se efectúe la **Escritura del Bien**
## Objetivo
- Adjudica al comprador (de buena fe) **prioridad sobre derecho de terceros** que tenga medidas cautelares sobre el inmueble

ej: Si tengo boleto de compra-venta y un tercero quiere embarga el inmueble, yo tengo prioridad para obtener dicho bien por sobre el 3ro. 
### Condiciones
1. Vendedor es titular registral (heredero o representante)
	- **Inmueble a nombre del vendedor**
2. **Comprador pagó 25%** (como mínimo) del precio anterior a la traba cautelar 
3. Boleto tiene fecha cierta 
4. Adquisición con publicidad suficiente (registraal o posesoria) 
___
# Medida cautelar 
- Por orden de juez
- Para **asegurar patrimonio de deudor** para que acreedor pueda cobrarse
