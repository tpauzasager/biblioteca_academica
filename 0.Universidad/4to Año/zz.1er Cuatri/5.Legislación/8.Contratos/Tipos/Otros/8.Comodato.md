# Definición 
- Comandante obliga a entregar al comodatario una cosa no fungible (única en el mundo)
- Le transfiere derecho de uso
- Si no tienen plazo -> Comodante puede reclamar restitución en cualquier momento
## Características
- Bilateral
- Gratuito
# Obligaciones 

| Comodante                          | Comodatario                          |
| ---------------------------------- | ------------------------------------ |
| Recibir cosa al finalizar contrato | Restituir la misma cosa al finalizar |
