| Nombre       | Descripción                                                                                                                         |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------- |
| Conmutativos | Las ventajas o las pérdidas son conocidas de antemano por las partes (ej.: compraventa).                                            |
| Aleatorios   | Las ventajas o pérdidas dependen de un acontecimiento futuro incierto y no pueden conocerse a priori (ej.: juego, apuesta, seguro). |
