| Nombre            | Descripción                                                                                                                                                             |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| A Título Gratuito | Asegura a una de las partes alguna ventaja sin contraprestación (no debe dar algo a cambio de lo que recibe) a su cargo (ej.: comodato, donación, depósito).            |
| A Título Oneroso  | Para que una de las partes intervinientes pueda recibir una ventaja, debe realizar una contraprestación (le debe dar algo a cambio a la otra parte) (ej.: compraventa). |
