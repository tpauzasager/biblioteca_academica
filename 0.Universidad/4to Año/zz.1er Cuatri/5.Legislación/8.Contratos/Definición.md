# Definición 
- Acto jurídico por el cual dos o más partes manifiestan consentimiento para la creación de una obligación entre ellos (relación jurídica)
# Principios 
| Nombre                           | Definición                                                                                                                 |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| Libertad de contratación         | Partes son libres de celebrar contrato (siembre que esté dentro de ley)                                                    |
| Efecto Vinculante                | Todo contrato válido es obligatorio para las partes<br>Modificación permitida por acuerdo de partes o supuestos por la ley |
| Buena Fe                         |                                                                                                                            |
| Caracteres de las normas legales | Normal legales supletorias se aplican cuando contrato no toma en cuenta ciertas condiciones                                |
| Derecho de propiedad             | Derechos del contrato puede ser considerados como parte del patrimonio del contratante                                     |
___
# Elementos

| Elemento       | Definición                                              |
| -------------- | ------------------------------------------------------- |
| Consentimiento | De ambas partes<br>Oferta y aceptación (u contraoferta) |
| Objeto         | Cosas o hechos sobre los que tiene efecto la prestaicón |
| Forma          | Depende de ley                                          |
| Cuasa          | Fin que se le va a dar al contrato                      |
