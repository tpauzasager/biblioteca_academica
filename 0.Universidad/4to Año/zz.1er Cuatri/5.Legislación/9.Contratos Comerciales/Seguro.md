# Definición 
- Aseguradora se obliga a resarcir (por prima o cotización) daño o cumplir prestación 
Finalidad
	- Transladar riesgo del asegurado al asegurador 
	- Cuando ocurre siniestro, consecuencias asumidas por él
## Partes

| Parte              | Descripción                                                 |
| ------------------ | ----------------------------------------------------------- |
| Tomador del seguro | Quien suscribe el contrato con el asegurador                |
| Asegurador         | Otorga póliza                                               |
| Asegurado          | Persona que tiene interés en conservar cosa asegurada       |
| Beneficiario       | Quien recibe indemnización <br>Aun no es parte del contrato |
## Caracteristicas

| Caracteristica | Descripción                                                             |
| -------------- | ----------------------------------------------------------------------- |
| Consensual     | Se prefecciona en momento de celebración                                |
| Aleatorio      | Objeto depende de hecho futoro e incierto                               |
| Adhesión       | Asegurador establece todas las cláusulas<br>Asegurado no puede negociar |
| Bilaterla      |                                                                         |
# Elementos 

| Elemento         | Definición                                  |
| ---------------- | ------------------------------------------- |
| Prima/Cotización | Dinero que paga el asegurado                |
| Indemnización    | Dinero que paga asgurador (en caso de daño) |
| Riesgo           |                                             |
| Póliza           | Instrumento del contrato<br>Por escrito     |
# Obligaciones

| Asegurador                                          | Tomador del seguro | Asegurado                             |
| --------------------------------------------------- | ------------------ | ------------------------------------- |
| Indemnizar si ocurre siniestro                      | Pagar prima        | Probar que prima fue pagada           |
| Afrontar téncia y económicamente el siniestro       |                    | Denunciar siniestro en timepo y forma |
| Obligaciones comienzan a las 12hs del día celebrado |                    | No agravar riesgos                    |
