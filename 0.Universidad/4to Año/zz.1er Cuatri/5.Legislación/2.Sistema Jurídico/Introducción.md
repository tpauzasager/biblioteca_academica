# Supremacía constitucional 
- Garantiza que la Constitución Nacional sea reconocida y ningún poder (legislativo, ejecutivo, judicial) puede dictar normal que la contradigan 

# Ley inconstitucional
- Una ley que no cumpla con CN debe ser decretada como inconstitucional (derogarla)
- Declaración de inconstitucionalidad -> Evita su aplicación 
- Derogación -> Solo por el órgano legislativo que la creo
