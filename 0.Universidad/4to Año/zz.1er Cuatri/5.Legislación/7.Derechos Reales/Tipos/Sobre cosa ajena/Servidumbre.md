# Definición 
- Entre dos inmuebles 
- El titual del inmuebe dominantes tiene dominio sobre la totalidad o una parte del otro inmueble sirviente ajeno. 


# Clasificación 
| Nombre   | Descripción                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------- |
| Positiva | Derecho de uso activo (para realizar ciertos actos)                                                        |
| Negativa | Obligar a dueño sirviente a no hacer algo <br>(construir otro piso más alto)                               |
| Real     | A favor de una propiedad (no de una persona)<br>ej: Campo que tiene derecho de paso a traves de otro campo |
| Personal | A favor de una persona<br>En caso de duda, se presume personal                                             |
| Forzosa  | Se impone por ley o necesidad                                                                              |
