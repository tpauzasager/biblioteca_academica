## Concepto 
- Coexistencia de propiedad privada y común dentro de una misma cosa (edificio de departamentos, tiene departamentos y escaleras, ascensores, etc) 
- Cada propietario tiene dominio exclusivo de su unidad privada
- Cada propietario comparte la copropiedad de áreas comunes (escalera, ascensor, etc) 

## Constitución
- Los condóminos deben redactar el "Reglamento de propiedad horizontal" que debe inscribirse en el registro inmobiliario

## Unidad funcional
- Este derecho se determina sobre la unidad funcional (dptos, etc) aprovechables, que tengan independencia funcional y comunicación con la vía pública

## Cosas y partes comunes
- Nadie tiene derecho exclusivosobre las partes comunes 
- Cada propietario puede usar las cosas y partes comunes sin perjudicar a otros propietarios

# Ejemplos
| Tipo                      | Ejemplos                                                  |
| ------------------------- | --------------------------------------------------------- |
| Necesariamente comunes    | terreno, pasillos, ascensores, artefactos e instalaciones |
| Comunes no indispensables | Piscinas gimnasio, lavadero                               |

# Documentos de adquisición 
- Al adquirir una UF (unidad funcional), el vendedor debe otorgar:
	- Escritura traslativa de dominio 
	- Certificación de libre deuda
	- Reglamento de copropiedad

# Obligaciones
- El propietario de la propiedad horizontal debe contribuir al pago de expensas (gastos necesarios para mantener el buen estado de seguridad, comodidad y decor) comunes y extraordinarias. El monto (porcentaje de las expensas) lo determina el Reglamento de Copropiedad.
