- "Dominio perfecto es el derecho real que otorga todas las facultades de usar, gozar y disponer material y jurídicamente de una cosa, dentro de los límites previstos por la ley. El dominio se presume perfecto hasta que se pruebe lo contrario."

# Diferencias

| Nombre         | Definición                                  | Ejemplos                      |
| -------------- | ------------------------------------------- | ----------------------------- |
| Propiedad      | Concepto más amplio                         | Prop intelectual (no es cosa) |
| Posesión       | Actua como dueño (sin serlo necesariamente) |                               |
| Tenencia       | Hace uso limitado                           | Alquiler                      |
| Dom Imperfecto | Sometido a condición                        |                               |

# Diferencia con "Propiedad"
- Propiedad es algo más amplio
- **Se puede tener propiedad de algo que no es una cosa** (i.e. propiedad intelectual) 
# Diferencia con "Posesión"
- **Actua como dueño** 
- Poseedor ejerce poder sobre la cosa sea o no dueño de esta
- Dominio = derecho real | Posesión = hecho jurídico
# Diferencia con "Tenencia" 
- Tenedor **ejerce poder** sobre la cosa **como representante del poseedor** 
# Dominio Imperfecto
- Si está sometido a condición, plazo resolutorios o cosa está gravada con cargas reales. (buscar info)
