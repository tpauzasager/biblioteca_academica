# Forma
- Libre
# Prueba de existencia del contrato
- Para probarse la existencia, debe probarse la Relación de Dependencia (por cualquier medio)
- No es necesario el contrato formal
# Casos especiales

| Caso                       | Descripción                                                                                                                |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| Socio empleado             | Es parte de la sociedad y realiza actividad en forma personal y habitual                                                   |
| Auxiliar                   | Si al trabajo se le permite tener auxiliares<br>Estos considerados como relación directa con empleador (salvo excepciones) |
| Contratista/subcontratista | Cuando empleado hace de empleador <br>Este nuevo empleado es considerado empleado directo de empresa principal             |
# Plazo
- Default => Indefinido 
- Si se aclara (expresamente) => Por un tiempo determinado 
- Modalidad de tareas deben justificar la temporalidad del trabajo
# Período de prueba
- **Dentro del contrato laboral**
- Máx 3 meses (desde el inicio) 
- Cualquiera de las partes puede exitngir relación sin penalización
