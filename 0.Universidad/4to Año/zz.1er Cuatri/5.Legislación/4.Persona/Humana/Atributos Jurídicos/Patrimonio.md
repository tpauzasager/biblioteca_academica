# Definición 
- Conjunto de bienes materiales (cosas) e inmateriales (derechos) suceptibles de calor económico que posee una persona 

# Características 
| Nombre                           | Definición                                                                                  |
| -------------------------------- | ------------------------------------------------------------------------------------------- |
| Necesario                        | Todas las personas tienen (aunque no tenga más que deudas)                                  |
| Inalienable                      | Se pueden enajenar las cosas individiualmente, pero no la totalidad del patrimonio como tal |
| Universalidad jurídica           | Entidad abstracta independiente de los elementos que la componen                            |
| Garantía común de acreedores<br> | Todos los bienes que tenga un deudor son garantía para saldar cualquier deuda               |

# Clasificación 
| Nombre        | Definición                                                                                                 |
| ------------- | :--------------------------------------------------------------------------------------------------------- |
| Personales    | Relacion entre personas<br>Facultad del acreedor para exigir a un deudor el cumplimiento de una perstación |
| Reales        | Conceden al titular poder inmediato sobre una cosa                                                         |
| Intelectuales | Derecho exclusivo para explotación económica por cualquier medio                                           |
