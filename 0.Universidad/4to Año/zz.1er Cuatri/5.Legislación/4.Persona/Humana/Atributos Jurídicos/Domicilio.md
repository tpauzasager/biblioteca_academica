# Definición 
- Asiente o sede de la persona 
- Permanente
- Donde realmente vivo o designado por ley para ejercer derechos y obligaciones 

# Clasificación 
| Tipo     | Definición                                                                                                                                                                                                                  |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| General  | - Donde persona reside habitualmente<br>- Voluntario, mutable e inviolab                                                                                                                                                    |
| Legal    | - Ley presume que persona reside permanentemente allí para ejercicio de obligacio                                                                                                                                           |
| Especi - Elegido por partes de un contrato<br>- Para ejercer derechos y relaciones jurídicas de dicho contrato<br>- Ej: Para contrato estableces domicilio en un estudio de abogados => notificaciones se envían a dicho estudio  dio  |
