# Definición 
- Aplicable a: **Potencia, tensión, corriente**

# Relativa
|           | Pérdida                     | Ganancia                    |
| --------- | --------------------------- | --------------------------- |
| Potencia  | $P(dB)=10·log(P_1/P_2)$     | $G(dB)=10·log(P_2/P_1)$<br> |
| Tension   | $P(dB)=20·log(V_1/V_2)$<br> | $G(dB)=20·log(V_2/V_1)$<br> |
| Corriente | $P(dB)=20·log(I_1/I_2)$<br> | $G(dB)=20·log(I_2/I_1)$<br> |

# Absoluta
$$dBW = 10.log(P/1W)$$
$$dBmW = 10.log(P/1mW)$$
$$dBu = 20.log(P/0,775v)$$
# Pasaje
$$P[mW] = 1mW.10^{P[dBm]/10}$$
$$V[mW] = 1mW.10^{V[dBm]/20}$$
