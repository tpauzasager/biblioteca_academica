# Definición 
Uso
	Mantener multiples comunicaciones en simultaneo por el mismo canal 

# Tipos Multiplexación 

| Tipo  | Modifica                                                             |
| ----- | -------------------------------------------------------------------- |
| FDM   | Frecuencia (f)                                                       |
| TDM   | Tiempo (t en T)                                                      |
| CDM   | Aplica XOR (?)                                                       |
| STDMA | Usa estadistica para dar más tiempo a canales que más info transmite |
| WDM   | División de longitud de onda                                         |


FDMA, TDMA, OFDMA, CDMA -> Para acceso múltiple (!= multiplexación)
