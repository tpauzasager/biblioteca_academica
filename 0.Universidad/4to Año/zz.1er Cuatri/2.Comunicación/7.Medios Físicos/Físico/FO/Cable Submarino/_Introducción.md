# Estructura

| Capa             | Definición                                                                                   |
| ---------------- | -------------------------------------------------------------------------------------------- |
| Núcleo           | Contiene fibras ópticas                                                                      |
| Polietileno      | Aislante para las FOs                                                                        |
| Tubo de cobre    | Para transmitir corriente eléctrica que alimenta repetidores y monitorear estado del sistema |
| Alambre de acero | Armazón para mayor resistencia a quebraduras                                                 |
| Polietileno      | Imperbeabilizante                                                                            |
## Elementos accesorios

| Elemento        | Proposito                                                                                |
| --------------- | ---------------------------------------------------------------------------------------- |
| Repetidor       | Amplificar señal para evitar atenuación                                                  |
| Branch Unit     | - Unidad de derivación<br>- Derivar cable de FO desde tramo principal a diferentes ramas |
| Caja de empalme | Para unir diferentes FO                                                                  |

# Tipos
Armado
	- Para poca profundidad
	- Alto nivel de protección (hay más riesgo en está profundidad)
Ligero
	- Gran profundidad
	- Menos protegidos
