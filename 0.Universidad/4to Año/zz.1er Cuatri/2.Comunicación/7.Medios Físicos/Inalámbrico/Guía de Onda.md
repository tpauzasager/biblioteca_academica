# Definición 
Medio de transmisión utilizado para transmitir señal electromagnética con alta frecuencia y mínima perdida (evitando dispersión) 
# Problema
Cuando onda en GHz viaja no por cobre 
- Radiación de energía
- Efecto pelicular
## Solucion
Usar Guías de Onda 
# Caracteristicas
- Tubo hueco 
- Longitud = [5;15]mts
- Sección permite propagación de onda electromagnética
- Hay atenuación (por humedad del aire) => Longitud menor
- Máyor pérdida a menos $\lambda$
# Propagación 
- A través de dieléctrico (aire tratado)
- Por medio de ondas electromagnéticas reflejadas en paredes
