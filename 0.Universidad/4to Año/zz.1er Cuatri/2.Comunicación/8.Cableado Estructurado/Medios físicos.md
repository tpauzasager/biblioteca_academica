# UTP
Definición 
	- 4 pares de hilos trenzados (para minimizar difonía entre pares)
	- Enlaces balanceados (dos conductores que llevan misma señal con polaridad invertida, para cancelar ruido)
	- Cada par forma camino eléctrico completo para transmisión de señal 
	- STP = UTP + blindaje
Caracteristicas
	- Menor atenuación 
	- Mayor respuesta en frecuencias 
	- Mayor inmunidad al ruido 
	- Disminución de **Paradinofía** (señal replicada en cable adyacente)
	- $Z_0 = 100 \Omega$   
Categoria depende de
	- Aleación del cobre 
	- Recubrimiento (de cada hilo y del cable total)
	- Trenzado del par (para evitar paradiania)
## Patch cord
Usos para conección de
	Estación de trabajo <-> Roseta de instalación 
	Patcher <-> Switch = Hub (en armario de cableado)
	Estación de trabajo <-> Switch
	Hub <-> Crossover de otro dispositivo
### Cable cruzado
Definición
	Patch cord que usa estandares diferentes en ambos extremos
Usos de conección 
	- Entre switchs
	- Entre estaciones de trabajo (para invertir parte receptora con parte transmisora) 
	- Estación de trabajo y servidor (sin switch)
