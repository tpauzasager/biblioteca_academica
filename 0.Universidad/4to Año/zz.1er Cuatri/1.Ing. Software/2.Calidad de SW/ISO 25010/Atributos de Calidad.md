| Nombre                  | Definicón                                                                                                           | Tipos                                                                                                                                                                           |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Adecuación Funcional    | Proporcionar funciones que satisfacen necesidades                                                                   | - Completitud funcional<br>- Correción Funcional<br>- pertinencia Funcional                                                                                                     |
| Eficiencia de Desempeño | Desempeño relativo a la **cantidad de recursos** utilizados                                                         | - Comportamiento temporal<br>- Utilización de recursos<br>- Capacidad                                                                                                           |
| Compatibilidad          | De varios sistemas (o componentes) para **intercambiar información** y llevar a cabo funciones compartiendo entorno | - Coexistencia<br>- Interoperabildad                                                                                                                                            |
| Usabildad               | Ser **entendido**, **aprendido**, **usado** y resultar **atractivo** al usuario                                     | - Capacidad para reconocer su adecuación <br>- Capacidad de aprendizaje<br>- Cap para ser usado<br>- Protección contra errores de usuario<br>- Estética de UI<br>- Accesibildad |
| Fiabilidad              | Desempeñar funciones                                                                                                | - Madurez<br>- Disponiblidad<br>- Toleracnaia fallos<br>- Capacidad de recuperación                                                                                             |
| Seguridad               | Protección de información y datos                                                                                   | - Confidencialidad<br>- Integridad<br>- No repudio<br>- Responsabilidad<br>- Autenticidad                                                                                       |
| Mantenibilidad          | Ser modificado efectiva y eficientemente (correctivo o perfectivo)                                                  | - Modularidad<br>- Reusabildad<br>- Analizabilidad<br>- Ser modificado<br>- Ser probado                                                                                         |
| Portabilidad            | Ser transferido efectiva y eficientemente de un entorno a otro                                                      | - Adaptabildad<br>- Ser instalado<br>- Ser reemplazado                                                                                                                          |
