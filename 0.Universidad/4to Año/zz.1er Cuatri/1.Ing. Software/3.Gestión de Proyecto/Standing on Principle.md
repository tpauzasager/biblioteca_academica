# Keys
- Cliente no siempre está correcto (pero hay que prestarle atención)
- Ir más allá de requerimientos del cliente 
- Buscar simples y efectivas soluciones (evitar overengineering)
- No prometer lo que no puedes cumplir 
- Evitar esconder límites del proyecto

## Clasificación 
| Nombre                  | Definición                    | Flexibilidad |
| ----------------------- | ----------------------------- | ------------ |
| Degree of Freedom (DoF) | Puede ser ajustado            | Alta         |
| Driver                  | Máxima prioridad del proyecto | Baja         |
| Constraint              | Inamovible                    | Nula         |
