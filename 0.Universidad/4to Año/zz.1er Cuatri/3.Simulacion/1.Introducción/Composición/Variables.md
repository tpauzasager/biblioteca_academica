# Exógenas
- Independientes / **Input** del modelo
- Predeterminadas y propocionadas independientemente del modelo 
- Actuan sobre el sistema
- No son afectadas por el sistema
## Datos
- No controlables, **no se pueden modificar** 
- Funciones de densidad de probabilidad 
- Se les aplica la **hipotesis de simplificación**
- Ej: Intervalo entre arribo de personas
## Variables de Control
- Controlables, se pueden **modificar** 
- Se ingresan por única vez al inicio de la simulación (no se pueden cambiar durante la misma)
- Ej: Nro cajas en un supermercado

___
# Endógenas
- Dependientes del sistema / **Output** del modelo 
## Estado
- Describen el estado del sistema en cada instante de tiempo 
- Son la **FOTO** del sistema 
- Son reflejo de todo cambio (**EVENTO**) en el stm 
- Cantidad de stock de unra mercadería (si vendo o repongo esa cantidad varía)
## Resultado
- Salida del sistema 
- Generada por **interacción entre var exógenas y de estado**
