# Definición formal
X simula a Y <=>
	- X e Y son sistemas formales
	- Y es sistema real
	- X aproximación al sistema real
	- Reglas de validez en X no están exentas de error

# Toma de decisiones
- Objetivo principal de la Simulación 
$$Y_i=R_i(U_i)$$
$Y_i=>\text{Resultados}$
$R_i => Realidad$
$U_i => \text{Alternativas decisorias}$

- Conocimiento de las relaciones (R_i) permite predecir resultado

# Razón de ser
- Ensayar sobre sistemas reales puede ser costo o llevar a destrucción del mismo 

# Metodología
- Se determina a partir de la clasificación de las variables (en etapa de analisis)
