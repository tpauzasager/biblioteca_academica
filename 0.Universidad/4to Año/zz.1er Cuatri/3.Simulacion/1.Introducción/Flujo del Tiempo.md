# Evento a Evento
## Propiedaes 
- Siempre mira hacia adelante (hacia el **siguiente evento**)
- Eventos pueden no generar eventos futurnos no condicionados 
# Metodología
- Se avanza hasta el siguiente evento (este podria estar en el mismo t, por lo que no se avanzaría nada)
- Se encarga de dicho evento 
- Repetir
