# TABLA DE EVNETOS INDEPENDIENTES
# Propiedaes 
- Una fila por evento independiente en el modelo 
- En 1ra y 2da fila columna se escribe el mismo evento 
- No se modifica a lo largo del tiempo 

# Columna anexa
- Contienen los valores de las variables de estado que tienen que tener para que ocurra dicho evento
