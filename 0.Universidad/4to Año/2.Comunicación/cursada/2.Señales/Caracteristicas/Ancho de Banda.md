# Definición
- Cantidad de términos de Fourier que se toman para pasar la señal
- Es un intervalo de frecuencias en el espectro de fourier 
- Por fuera de este ranto, la atenuación es muy grande 
___
# Calculo
### $$\Delta f=n·f_0$$
$f_0= 1/d = \text{Frecuencia fundamental}$
$FRP = 1/T = \text{Frecuencia de Repetición de Pulso}$
Cant de armonicos suficientes: 
$$n = T/d$$ 
Armonicos:
$$w_n=\frac{2\pi n}{T}$$
