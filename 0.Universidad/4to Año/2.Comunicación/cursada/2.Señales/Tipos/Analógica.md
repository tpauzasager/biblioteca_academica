# Definición
- Representadas por funciones continuas en el tiempo con imagen infinita
- **Información en forma de onda**
## Señal senoidal
$$S_{(t)}=A·sin(\phi+\omega·t)$$
### Parámetros
| Sim      | Nombre            | Definición                                    | Formula                    |
| -------- | ----------------- | --------------------------------------------- | -------------------------- |
| $\omega$ | Velocidad Angular |                                               |                            |
| $T$      | Periodo           | Tiempo de cada ciclo de la señal (frecuencia) | $2\pi / \omega$            |
| $\phi$   | Fase              | Desplazamiento inicial                        | $sin(x + \pi /2) = cos(x)$ |
| $A$<br>  | Amplitud          | Instantanea ($S_t$)o Máxima                   |                            |
## Longitud de onda
$$c=\lambda · f$$
$c=\text{Velocidad de la luz}$
$\lambda = \text{Longitud de onda}$
$f = \text{Frecuencia}$
