# Función 
- Filtra frecuencias que no estén en el rango determinado 

# Parámetros 
Ancho de banda
	Rango de frecuencias 
Frecuencias de corte
	Cota superior e inferior (f1, f2)
Selectividad
	Rojo es más selectivo, ya que filtra más cantidad de frecuencias (mayor pendiente)

# Comportamiento 

| Tipo      | Descripción                          |
| --------- | ------------------------------------ |
| **Ideal** | Selectividad del 100%                |
| **Real**  | Hay pendiente (selectividad != 100%) |
