# PDH
Jerarquía digital pleosincronica 

Caracteristicas
	- Se agregan bits de relleno para complensar sincronismo y indicar con que canal se comunica
	- La salida de un mux se usa como entreda de otro mux de siguiente orden (lo mismo para los demux)
E_0 -> 64 Kbps
E_1 -> 32.E_0 = 2Mbps
# SDH
Jerarquía digital sincrónica

Caracteristicas
	- Se inyecta directamente en trama 
	- Más eficiente
