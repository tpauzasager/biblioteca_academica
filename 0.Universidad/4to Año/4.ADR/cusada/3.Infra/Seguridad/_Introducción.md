# Seguridad de la Información 
Definición:
	Medidas preventivas y reactivas de sistemas tecnológicos para proteger info 
	Manter: 
		Confidencialidad -> Restringir acceso a lectura
		Disponibilidad -> Poder acceder en tiempo y forma
		Integridad -> Restringir acceso a escritura
# Sistemas de Gestión de Seguridad de Información
(SGSI)
Definición -> Gestionar programas, controles y políticas para mantener confidencialidad, disponibilidad e integridad de la info
Elementos:
	- Elementos administrativos
	- Políticas de seguridad
	- Organización y división de responsabilidades
	- Seguridad lógica
# Evento vs Incidente de Seguridad
Evento -> Ocurrencia identificada en el sistema que indica posible violación de seguridad
Incidente -> Evento que ocurrió y tiene prob de comprometer operaciones 
# Respuesta a Incidente 
1. Detener / Minimizar 
2. Investigar 
3. Restaurar
4. Reportar 
# Tipos de Ataques
Phishing - jacking - Malware - Extorsiones - Explotación de vulnerabilidad - Suplantación de identidad
# Seguridad Lógica
- Control de accesos
- Acceso solo a la info necesaria 
- No repudio
- Autenticación
# Firma Digital vs Electrónica
Digital -> Clave pública y privada 
Electrónica -> Datos asociados a otro conjunto de datos (archivo original) que carece de algún requisito para ser Digital
