# Firewall
Definición
	- HW o SW
	- Filtra trafico que pasa por la red hacia el servidor
	- A nivel de paquetes de datos (PDU)
___
# IDS - IPS
Intrusion Detection System 
	- Analiza trafico (que ya pasó por el fw) y informa si hay vulnerabilidad
	- No es bloqueante (solo genera aviso)
	- Capa 3 
Intrusion Protection System
	- IDS pero bloqueante 
	- Capa 4+
___
# WAF
Web Application Firewall

Definición
	- Detectar vulnerabilidades en aplicaciones web (i.e. sqli, etc.)
___
# VPN
Virtual Private Network
___
# SIEM 
Security Information and Event Management

Definición
	- Centraliza logs de toda la infraestructura (lo que va haciendo cada cliente)
	- Determina si un comportamiento no está permitido
