# Primarias

| Acronimo | Variable       | Definición                                                                |
| -------- | -------------- | ------------------------------------------------------------------------- |
| PV       | Valor planeado | Lo que planifique que iba a poder hacer hasta este momento                |
| AC       | Costo real     | Costo real hasta el momento para realizar lo que se hizo hasta el momento |
| EV       | Valor ganado   | Lo que verdaderamente hice hasta el momento                               |

# Secundarias

| Acronimo | Variable                  | Definición                                               | Formula         |
| -------- | ------------------------- | -------------------------------------------------------- | --------------- |
| BAC      | Presupuesto del proyecto  | Costo total del proyecto previsto inicialmente           |                 |
| ETC      | Estimación para finalizar | Estimación del costo del trabajo restante para finalizar |                 |
| EAC      | Estimado a la conclusión  | Estimación desde actualiadad hasta finalización          | $AC+ETC$        |
| VAC      | Variación a la conclución |                                                          | $\|BAC - EAC\|$ |
