# Abastecimiento 
- Proceso por el cual una organización adquiere bienes o contrata servicios
# Gestión del abastecimiento 
- Utilizar recursos disponibles de manera efectiva y eficaz 
- Para mejorar proceso de compra de BS
# Etapas
## Definición de requerimientos
- Detección de necesidades 
- Definición de productos o servicios para satisfacer necesidades
- **Convertir: Necesidad de usuario -> Requerimientos para proveedores**
## Selección del mecanismo de compra 
- Una vez definido que se compra
- Se define mecanismo para adquirir el BS
- Mecanismos más comunes:
	- Convenio / Marco
	- Licitación Pública
	- Licitación Privada
	- Trato Directo
## Llamado y recepción de propuesta
- Formato depende del mecanismo de compra
## Evaluación de propuesta
- Una vez adquiridas las propuestas de oferentes
- Analizar si satisfacen especificaciones
- Definicir previamente el método de analisis
## Adjudicación de ofertas 
1. Cierre de evaluación y selección del proveedor
2. Formalización de acuerdos de facturación (entre otros)
3. Adjudicación debe ser documentada 
## Recepción del producto 
4. Recepción y verificación del BS (en base a bases y condiciones de la solicitud)
5. Determinación de conformidad del BS entregado 
## Seguimiento y monitoreo de la compra 
6. Evaluación de proveedores 
7. Revisión periódica 
8. Tener claro mecanismos de garantía
9. Ordenar, tabular info relevante para futuras compras
