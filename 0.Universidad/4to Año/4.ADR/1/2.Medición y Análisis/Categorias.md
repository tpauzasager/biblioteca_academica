# Compliance (Cumplimiento)
_La ausencia de evidnecia no es evidnece de ausencia_
- Determinan **si proceso se está realizando** según lo documentado 
- Ej:
	- % de cambios liberados 
# Calidad
- **Que tan bien se hace algo**
- Ej:
	- % incidentes mal asignados / no cerrados
# Rendimiento
- Velocidad con la que está sucediendo
- Ej:
	- t medio de resolución 
	- % incidentes resulestos dentro de $\Delta t$ 
# Valor
- Medida de lo que genera el proceso 
- Quien está recibiendo el resultado del proceso 
- Ej:
	- Satisfacción usuario
