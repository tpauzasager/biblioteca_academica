
|                                                  |                          |                                                           |
| ------------------------------------------------ | ------------------------ | --------------------------------------------------------- |
| Valor Planeado                                   | PV                       | Cuanto iba a gastar para cierto momento                   |
| Costo real                                       | AC                       | Costo real de lo construido hasta el momento              |
| Valor Ganado                                     | EV                       | Aumento de valor del proyecto                             |
| ----                                             |                          |                                                           |
| Presupuesto del proyecto                         | BAC                      |                                                           |
| Estimación para finalizar                        | ETC                      | Cuanto me falta para terminar                             |
| Estimado a la conclusión                         | $EAC=AC+ETC=BAC/CPI$     | Estimación del total del proyecto desde donde estoy ahora |
| Variación a la conlusión                         | $VAC=BAC-EAC$            | Desvío del costo total                                    |
| ----                                             |                          |                                                           |
| Desempeño de cronograma                          | $SPI = EV/PV$            |                                                           |
| Desempeño de Costo                               | $CPI=EV/AC$              |                                                           |
| ----                                             |                          |                                                           |
| Desempeño de costos para finalzar                | $TCPI=(BAC-EV)/(BAC-AC)$ |                                                           |
| Desempeño de tiempo requerido para finalizar<br> | $TSPI=(BAC-EV)/(BAC-PV)$ |                                                           |
