# Medición y Análisis
Definición:
	Medición es la reducción de la incertidumbre expresada cuantitativamente y basada en un ao más observaciones 

Proposito:
	Desarrollar y sostener las mediciones para satisfacer necesidades de info de gestión 

Tipos Métricas:
	- Vanidosas -> Maquilladas para parecer que todo va bien (para presentaciones de promoción)
	- Accionables -> Reas y crudas 

Caracteristicas:
	- Comparativa -> Tener unidad de medida 
	- Compresible -> Por que se realizó?
	- Guardar relación -> Contextualizarla

Categorías:
	- Cumplimiento -> Si se hace lo que se debe
	- Calidad -> Que tan bien se hace algo
	- Rendimiento -> Velocidad de realización
	- Valor -> ?

## OKR
Objective and Key Result

Definición 
	- Para crear compromiso y alineamiento a los objetivos
	- Para métodología ágil (se adaptan a los cambios)
Caracteristicas
	- Ágil
	- Simplicidad
	- Transparencia 
	- Ritmos anidados
	- Objetivos bidireccionales -> No usa modelo cascada
	- Objetivos ambiciosos
	- Desacoplamiento de recompensas -> Para que empleados no pierdan interes en poner OKRs ambiciosos (dando la oportunidad a que no los cumplan luego)
