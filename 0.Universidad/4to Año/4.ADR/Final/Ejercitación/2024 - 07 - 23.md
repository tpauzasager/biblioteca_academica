	1. En el ámbito de EVM desarrolle los siguientes puntos:
		1. ¿Es posible estimar el costo con el que finalizará el proyecto si hubo variaciones respecto de lo estimado? ¿Qué fórmula utilizará en los diferentes escenarios? Justifique sus respuestas.
			EAC(Estimated at Conclution) = AC + ETC(Estimated to Conclution) se utiliza para estimar en el que se habrá incurrido para finalizar el proyecto. Este tambien se utiliza para calcular dicho costo frente a variaciones durante la vida del proyecto. 
			Si el proyecto mantiene una variación constante: EAC = BAC /CPI
			Si el proyecto no mantiene variación constante: EAC = AC + BAC - EV 
			La estimación original fue muy mala y se hace una nueva: EAC = AC + Nueva Estimación
			
		2. Enuncie y describa qué variables/indicadores utilizaría para determinar si el proyecto está dentro de lo planificado. ¿Considera que a lo largo del proyecto las variables a considerar mantienen su misma importancia? Justifique su respuesta.
			CPI = EV / AC indica el rendimiento que ha tenido lo invertido en el proyecto hasta el momento. CPI < 1 => El proyecto está gastando más de lo esperado (hasta el momento).
			SPI = EV / PV indica el rendiimento en terminos de cronograma del proyecto, es decir, si el proyecto está avanzando conforme a lo planeado. SPI < 1 => El proyecto está yendo por detras del cronograma (tardó más en realizar las tareas hasta el momento).
			Estas variables mentienen su importanica a lo largo del desarrollo de todo el proyecto
