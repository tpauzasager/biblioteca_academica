# Plan de Contingencia y Recuperación 
Definición
	- Antes el riesgo de disrrupción, organización usa plan de contingencia
	- Ante disrrupción, organización usa plan de recuperación

Recovery Time Objective (RTO)
	Tiempo que puede estar caido el servicio sin afectar el negocio
Recovery Poin Objectiva (RPO)
	Cant de datos que puedo perder sin afectar el negocio
