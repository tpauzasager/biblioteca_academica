# Monitoreo vs Observabilidad
Monitoreo -> Actividad de supervisar un sistema
Observabilidad -> Capacidad de un sistema para ser evaluado a partir de inputs y outputs

# Definición 
- Es un requerimiento no funcional 
- Sistema puede ser observable, pero no monitoreado (si no se ven los datos de la observabilidad)

Objetivos:
	- Enterarme rápido de los fallos 

Métricas:
	TTD -> Time to Detect
	TTM -> Time to Mitigate 

Mediciones
	- Cantidad instancias (containers)
	- Uso de CPU
	- Response time 
	- Error rate
	- Apdex => Satisfacción de usuario (?)
	- Analisis de uso (para detectar anomalias)
	- Business core-metrics
