# Maquina Termica
## Definición 
- Sistema Termodinámico cerrado
- Evoluciona ciclicamente
- Parte del calor se desaprovecha (lo devuelve al medio)
- Se conserva la cantidad de energía 
$$\color{yellow}Q_1=Q_2+W$$

# Teorema de Carnot
- Existe un rendimiento límite (a través de evoluciones reversibles) para las máquinas térmicas 
![[Pasted image 20240829181813.png]]

$$\frac{T_2}{T_1}=\frac{Q_2}{Q_1}$$
$$\color{yellow}\eta_{~Tr}=1-\frac{T_2}{T_1} ~~~;(en °K)$$
$$\color{yellow}\varepsilon~_{Fr}=\frac{T_2}{T_1-T_2}$$
