# Politrópicas 
$c = cte$

| Tipo         | Caracteristicas                                                                                                                                                           |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Reversible   | - Pasa por puntos de estado de equilibrio <br/> - Infinitamente lenta <br/> Ecuación de estado vale en todo momento <br/> - En cámino inverso pasa por los mismos estados |
| Irreversible | - Muy rápida  <br/> -En camino inverso, no pasa por mismos estados <br/> - Pasa por estados de no equilibrio, hasta estado de equilibrio                                  |
