Transferencia del calor en solido
$$\color{yellow} P=\frac{\varDelta T}{R}$$
___
# Conducción
## Simetrías 
| Nombre     | Resistencia Térmica                                                       |
| ---------- | ------------------------------------------------------------------------- |
| Plana      | $$\color{yellow}R=\frac{L}{\lambda.A}$$                                   |
| Cilíndrica | $$\color{yellow} R=\frac{ln(r_f/r_i)}{\lambda.2\pi.L}$$<br>               |
| Esférica   | $$\color{yellow}R=\frac{1}{\lambda.4.\pi}·(\frac{1}{R_i}+\frac{1}{R_f})$$ |
## Sistema de Superficies 
| Serie               | Paralelo                                |
| ------------------- | --------------------------------------- |
| $$R_{eq}=\sum R_i$$ | $$\frac{1}{R_{eq}}=\sum \frac{1}{R_i}$$ |
___
# Convección 
$$\color{yellow}R=\frac{1}{h·S}$$
S = superficie de contacto
h = coeficiente de convección = $[h]=W/m²·g$ 
___
# Radiación 
$$\color{yellow}P=\varepsilon·\sigma·S·(T_i^4-T_\infty^4)$$
S= superficie de contacto 
$\varepsilon$ = emisividad (0 < e < 1)
$\sigma=5,67·10^{-8}~W/m²·k^4$
