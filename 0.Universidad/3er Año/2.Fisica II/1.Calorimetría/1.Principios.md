# 1.
Si se trasnfiere calor y energía, el sistema acumula energía (energía interna)
$$\color{yellow}Q_{M\rightarrow S}=\varDelta U+W_{S\rightarrow M}$$
$$\color{yellow}Q_{M\rightarrow S}=c_p~.~n~.~\varDelta T$$
- Si evolución es cerrada $\rightarrow \color{orange}U_f=U_i$ (sin depender de los estados intermedios)
- Q < 0 => Calor entregado
- W < 0 => Trabajo recibido
