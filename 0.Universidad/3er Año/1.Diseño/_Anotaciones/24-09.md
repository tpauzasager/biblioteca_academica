Función main
	Se tiene uno que es el principal del proyecto y se levante todo el contexto de javalin (para hcer consultas)
	- Levantar framework
		`Javalin main = JavaLin.create().start(8081)` main == nombre de la clase que contiene el método main
	- Crear endpoint:
		`main.get("/prueba", ctx -> ctx.result("Hola mundo!"));`

Consola de desarrollo:
Elements: 
Consola: Errores de javascript
Network: Request que se hacen

EndPoint -> Devuelve json
Ruta -> Devuelve contenido html

Acceder a las Query params:
	`main.get("/saludo", ctx -> {ctx.result("hola " + ctx.queryParam("nombre"));})`
	ctx.queryParam puede devolver nulo
	
Acceder a los Path/Rute params:  
	`main.get("/saludo-para/{nombre}", ctx -> ctx.result("HOla" + ctx.pathParam("nombre")))`

**NO** usar id autoincrementales si se van a pasar por parametros en la ruta/query, hay que usar un uuid (que es eso?)
