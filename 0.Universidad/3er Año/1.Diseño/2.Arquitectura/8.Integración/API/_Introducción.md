Definición:
	- Application Programming Interface 
	- Es una interfaz que se comunica con distintas aplicaciones con las que comparte datos 	
	- Permite que un sistema se comunique con otro sin saber como se implementar la solución internamente

Tipos:
	- Publica / Privada (requieren api key)
	- gRPC (Remote Procedure Calls) -> Google
	- SOAP (Simple Object Access Protocol)
	- REST (Representational State Transfer)
