Definición:
	Presenta los datos y su forma de interactuar en formato adecuado para el usuario

Objetivos:
- Posee lógica para poder represntar los datos de forma más amigable para el usuario
- Envía acciones del usuario al controlador
- Solicita actualizaciones a través del cotroller 
- Comunicación Vista-Controller se hace mediante objetos de tranferencia (DTO - Data Transfer Objects)
