# Cookies
Definición:
	- Pequeño archivo creado por Servidor y guardado en Cliente
	- Viajan en cada request del cliente
	- Guardan preferencias e info útil para las pág que visita el usuario
	- Sirven para que servidor identifique al usuario que realiza la request
