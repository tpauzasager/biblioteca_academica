Definición:
	- Contexto que refiera al HW y SW donde se ejecuta una aplicación 
	- No necesariamente son todos los componentes de un sistema
___
# Lugar de ejecución
Mismo Entorno:
	- Comunicación sencilla
	- Reducir al mínimo el **grado de acoplamiento**
	- Evitar **ruptura de encapsulamiento** del componente integrado
	- Integración debe ser testeable

Diferente Entorno:
	- Pensar por cual mecamismo se comunican (si no está definida la interfaz)
	- Si ya está definida la interfaz -> Hay que acoplarnos a las exigencias
___
# Testing con Integraciones
- Interfaz **saliente** debe ser **mockeada** (de no hacerlo, se llamaría cada vez que se testea el componente)
