# Definición
Aumento sostenido y generalizado del nivel de precios de BS
- Proceso Inflacionario: Cuando inflación desborda toda posibildad de control y planeamiento económico 

# Tipos
| Nombre        | Definición                         |
| ------------- | ---------------------------------- |
| Moderada      | Lenta                              |
| Galopante     | Incremento a tasa de 2 o 3 dígitos |
| Iperinflación | Hasta el 1000% anual               |

# Causas
| Nombre        | Definición                                                     |
| ------------- | -------------------------------------------------------------- |
| De Demanda    | Aumenta demanda sin que aumente la producción                  |
| De costos     | Aumenta costo de obra o mat prima                              |
| Autocontruida | Agentes prevén aumentos futuros de precios y ajustan conducta  |
