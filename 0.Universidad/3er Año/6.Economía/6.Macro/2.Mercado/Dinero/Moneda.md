# Dinero
- Activo o bien aceptado como medio de cambio o medición de valor 
## Tipos

| Material                               | Bancario                           |
| -------------------------------------- | ---------------------------------- |
| Medio de intercambio<br>Billete/Moneda | Depositos en entidades financieras |
## Caracteristicas
| Nombre            | Descripción                         |
| ----------------- | ----------------------------------- |
| Durabilidad       | No se deteriora en poco tiempo      |
| Estabilidad       | No fluctua su valor                 |
| Transportabilidad | Facil de transportar                |
| Divisibilidad     | Dividir en pequeñas partes          |
| Homogénea         | Unidades tienen mismo valor         |
| Oferta limitada   | Para que retenga el valor económico |
