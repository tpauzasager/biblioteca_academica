# Ley de rendimientos decrecientes
Si se mantienen los demás factores de producción cte, el aumento de uno generara un aumento en la producción con derivada 2da negativa (a medida que se agrega más de dicho factor, la producción aumento menos que el agregado anterior)

# Variables

| Precio Bien             | Precio Factores               | Tecnología                 | Expectativas    |
| ----------------------- | ----------------------------- | -------------------------- | --------------- |
| + precio BS => + oferta | + precio factores => - oferta | disminuye costo producción | aumenta demanda |
# Funcionamiento

| Aumento                                              | Disminución                                         |
| ---------------------------------------------------- | --------------------------------------------------- |
| - precio factores<br>+ teconología<br>+ expectativas | + precio factores<br>- tecnología<br>- expectativas |
