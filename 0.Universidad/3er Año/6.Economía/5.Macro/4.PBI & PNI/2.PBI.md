|         | Nominal                                                                                                                                                                                                                                                                                 |
| ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Precios | Corrientes                                                                                                                                                                                                                                                                              |
| Función | Evaluar actualidad del sist productivo                                                                                                                                                                                                                                                  |
| Calculo | Como flujo de gastos:<br>$$PBI_{nom}=Consumo+Inversión+Gasto~público+(Exportación-Importación)+Variación~de~existencias$$<br>Como flujo de rentas:<br>$$PBI_{nom}=(\sum{\text{remuneraciones de factores productivos}})+(Impuestos~indirectos-subsidios)+Amortizaciones+Gasto~público$$ |

|         | Real                                     |
| ------- | ---------------------------------------- |
| Precios | Constantes                               |
| Función | Evaluar evolución de producción          |
| Calculo | $$PBI_{real}=(PBI_{nom}/Deflactor)*100$$ |
___
# Deflactor

|         | IPI                                                                     | IPC                                                                        |
| ------- | ----------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| Nombre  | Indice de Producción Industrial                                         | Indice de Precios al Consumidos                                            |
| Mide    | - Evolución de actividad productiva<br>- Contempla variación de precios | - Variación de precios de la canasta básica<br>- Inflación y costo de vida |
| Enfoque | Productor                                                               | Consumidor                                                                 |
| Calculo | $$IPI=(Q_f/Q_i)*100$$                                                   | $$IPC=(P_f/P_i)*100$$                                                      |
