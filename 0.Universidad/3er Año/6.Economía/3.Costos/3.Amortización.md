# Calculo 
| Tipo        | Gráfico                               | Calculo                                   |
| ----------- | ------------------------------------- | ----------------------------------------- |
| Constante   | ![[Pasted image 20250128190413.webp]] | $C_i=\frac{Val~Amortizable}{N}$           |
