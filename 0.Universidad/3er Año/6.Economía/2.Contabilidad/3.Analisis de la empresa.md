# Económico
| Índice           | Formula            | Descripción                                                                     |
| ---------------- | ------------------ | ------------------------------------------------------------------------------- |
| ROA              | $Resultado/Activo$ | Si tamaño de empresa es acompañado con mantenimiento de niveles de rentabilidad |
| ROE              | $UAII/PN$          | Rendimiento de fondos propios (impacta en cotización)                           |
| Apalancamiento   | $ROE/ROA$          | Relación entre costo de recursos ajenso y rentabilidad obtenida de ellos        |
| Margen de ventas | $Resultado/Ventas$ | Que parte de las ventas son ganancias                                           |
| Rotación de X    | $Ventas/X$         |                                                                                 |

# Financiero
| Índice                                    | Formula                                                            | Descripción                             |
| ----------------------------------------- | ------------------------------------------------------------------ | --------------------------------------- |
| Capital de trabajo<br>(Fondo de maniobra) | $AC-PC$                                                            | Capacidad de saldar pagos a corto plazo |
| Solvencia                                 | $Total = A/P$<br>$Corto~Plazo=AC/PC$                               |                                         |
| Endeudamiento                             | $Bancaria=Dudas~banco/P$<br>$Total=P/A$                            |                                         |
| Liquidez                                  | $Corto~Plazo=(CajaBanco + Créditos)/PC$<br>$Absoluta=CajaBanco/PC$ |                                         |
