# Económica

| Indice                       | Formula                                   | Definición                                         |
| ---------------------------- | ----------------------------------------- | -------------------------------------------------- |
| Rotaciones                   | $$R_a=\frac{Ventas}{Q_a·P_a}$$            | Eficiencia en uso de activos                       |
| Margen de ventas             | $$Margen_a=\frac{Utilidad~Neta}{Ventas}$$ | Porcentaje de ventas que son ganacia               |
| ROE                          | $$ROE=\frac{Utilidad~Neta}{PN_{prom}}$$   | Rentabilidad del capital de accionistas            |
| ROA                          | $$ROA=\frac{Utilidad~Neta}{Activo}$$      | Rentabilidad del capital propio                    |
| Palanca                      | $$Palanca=\frac{ROE}{ROA}$$               | Uso de financiamiento extre en operación           |
| Plazo de Inmovilización de i | $$\frac{Bienes_i*365}{Costo~de~Ventas}$$  | Tiempo durante el cual el bien i no se pudo vender |
___
# Financiera

| Indice        | Formula                                               | Definición                                      |
| ------------- | ----------------------------------------------------- | ----------------------------------------------- |
| Solvencia     | $$S=\frac{Activo}{Pasivo}$$                           | Capacidad de cumplir obligaciones a largo plazo |
| Liquidez      | $$L_{absoluta}=\frac{Disponibilidades+Creditos}{PC}$$ | Capacidad de cumplir obligaciones a corto plazo |
| Endeudamiento | $$E=\frac{PE}{A}$$                                    | Cant de activos financiados por dueda           |
| Cobranzas     | $$Cob=\frac{Creditos}{Ventas}$$                       | Capacidad de cobro de cuentas por cobrar        |
___
# Proyectos
| Inidice | Formula                                                           | Definición                                                |
| ------- | ----------------------------------------------------------------- | --------------------------------------------------------- |
| ROI     | $$ROI=\sum\frac{(Utilidad~AII - Honorarios)_i}{Inversión~total}$$ | Rentabilidad de la inversión (en relación a su costo)     |
| VAN     | $$VAN_{(X)}=\sum\frac{Flujo_i}{(1+X)^i}$$                         | Mide cuanto valor agrega el proyecto en términos acutales |
