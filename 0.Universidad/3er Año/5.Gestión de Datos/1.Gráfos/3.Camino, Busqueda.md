Camino:
- Existe vinculo directo o indirecto entre ambos nodos
- No se considera el sentido 
Paso: 
- Existe camino entre ambos, pero con un sentido preestablecido
- Si se considera el sentido 
Ciclo:
- Paso donde origen =  destino
- Bucle == Ciclo con un solo vértice
