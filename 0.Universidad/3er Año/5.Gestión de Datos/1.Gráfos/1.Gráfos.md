# Definición 
G = (V, A)
- V -> vértices, nodos
- A -> aristas, relaciones entre pares de vértices 

# Objetivo
***Modelizar un problema específico*** a través de un modelo abstracto que establezca elementos que participan en el problema (vértices) y las relaciones que existen entre estos (aristas). 
Para ***simplificar la solución del problema***  
# Clasificación 
Según sentido
- Dirigido: Los aristas tienen sentido (relación unidireccional)
- No dirigido: Relaciones bidireccionales 

Grado
- Positivo: Aristas que salen del nodo
- Negativo: Aristas que llegan al nodo

| Nombre             | Carácteristica                                                                                                              |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------- |
| Libre              | No tiene aristas                                                                                                            |
| Completo           | Cada vértice está conectado a todos los demás vértices                                                                      |
| Regular de grado g | Todos los vértices tienen grado g                                                                                           |
| Simple             | Existe solo un único arista que une a dos vértices                                                                          |
| Complejo           | Existe más de un arista que une dos vértices                                                                                |
| Conexo             | Todo par de vértices está conectado po un camino                                                                            |
| No conexo          | Un grupo de vertices no está conectado al resto de vértices                                                                 |
| Complementario     | Complementario a G si posee mismo V y el complementario de A                                                                |
| Restricto          | Relación no debe cumplir con propiedad de **reflexividad (bucles), simetría (relaciones en ambos sentidos), transitividad** |
| Irrestricto        | No hay restricciones a las relaciones                                                                                       |
