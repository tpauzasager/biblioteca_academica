# Definición 
Uno o ningún predecesores y uno o níngun sucesor
Ventaja por sobre gráfos: Gracias a limitaciones por ser restricto unívoca--> Es más facil administrarlas 

# Bi-Unívocas 
Cada vértice tiene uno o ningún predecesor o sucesor 

| Nombre | Características                                                                                                                                                                                                                            |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Lista  | - Dinámica abierta (no hay restricción a la hora de agregar un elemento)<br>- Lineal: Último nodo apunta a null<br>- Circular: Último nodo apunta al primero<br>- Doble enlazada: Cada nodo tiene puntero al anterior y siguiente nodo<br> |
| Pila   | - LIFO (Las In Fist Out)                                                                                                                                                                                                                   |
| Cola   | - FIFO (First In First Out)                                                                                                                                                                                                                |

# Unívocas
Tiene uno o más sucesores, pero solo un predecesor
