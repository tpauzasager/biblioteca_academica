# Dinámica vs Estática

| Carácteristica         | Dinámica                                        | Estática                                                              |
| ---------------------- | ----------------------------------------------- | --------------------------------------------------------------------- |
| Espacio utilizado      | Cambia en función del gráfo                     | Se establece espacio fijo (contempla cant máx de relaciones posibles) |
| Tamaño del espacio     | Requiere uso de punteros (que ocuparán espacio) | Utiliza concepto de contigüidad (como en vectores)                    |
| Velocidad de operación | Peor, requiere tiempo para crear nueva relación | Mejor, el espacio ya está asignado (no requiere pedir memoria)        |
# Diferencias
Eficiencia en utilización de espacio:
- Mientras más relaciones haya, peor será el dinámico (porque habrán más punteros almacenados)
- Gráfos dispersos (muchos nodos y pocas relaciones) --> Dinámica
- Gráfo denso (pocos nodos y muchas relaciones) --> Estática
