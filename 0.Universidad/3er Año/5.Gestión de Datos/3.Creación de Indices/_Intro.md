Estructura auxiliar para acceder más rápido a la información 
Para poder mantener ordenados los datos (en funsión de una clave)

# Tipo de Acceso
| Nombre              | Definición                                             |
| ------------------- | ------------------------------------------------------ |
| Secuencial          | En funsión del modo en que ingresaron los datos        |
| Secuencial Indexado | En función de una clave                                |
| Random              | Directamente a una clave sin realizar ningún recorrido |
