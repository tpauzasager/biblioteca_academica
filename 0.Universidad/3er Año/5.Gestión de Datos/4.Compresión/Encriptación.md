# Encriptación 
Modificar contenido de un archivo para que mantenga contenido, pero no sea visible en formato tradicional
No puede cambiarse su tamaño ni espacio ocupado 

Procesos

| Nombre         | Descripción                                                                                               | Subtipos                                                                                                                  |
| -------------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| Desplazamiento | Desplazamiento (intercambio de posición) de bits de los caracteres en función de algún patrón             |                                                                                                                           |
| Reemplazo      | Reemplazar determinados caracteres en función de un patrón (fijo, variable o con intervención de usuario) | Fijo: Cada caracter se remplaza por otro definido de acuerdo a un patrón<br>Variable: En función de una clave del usuario |
| Mixto          | Se aplican ambos en un orden determinado                                                                  |                                                                                                                           |
