Definición:
	Técnicas algoritmicas para obtener información implícita en grandes bases de datos

Objetivos:
	Predecir tendencias y comportamientos

### Caracteristicas
| Caracteristicas de la BD | Descripción                                                  |
| ------------------------ | ------------------------------------------------------------ |
| Gran cant de columnas    | Cuantas más columnas en el DW meyor nivel de análisis del DM |
| Gran cant de filas       | Para disminuir errores de estimación                         |

| Caracteristicas del DM                | Descripción                                           |
| ------------------------------------- | ----------------------------------------------------- |
| Recolección de datos en gran escala   | Unificar BD dispoinbles                               |
| Alta tecnología y gran almacenamiento | Se necesita gran potencial de procesamiento y memoria |

# Ventajas
- Automatización de toma de decisiones en gran volúmen de datos generado por procesos tradicionales
- Permite dar prioridad de decisiones y acciones (indica factores con mayor incidencia)
- Genera modelos descriptivos (identificar patrones) y predictivos (expone relaciones no descubiertas)
