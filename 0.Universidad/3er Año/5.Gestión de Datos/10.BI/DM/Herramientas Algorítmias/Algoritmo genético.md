Definición:
	Técnicas de optimización basada en evolución 
	Usa combinación de genética, mutación y selección natural
___
Funcionamiento:
	Se tiene un conjunto de soluciones y se elijen las que minimicen la función objetivo, luego se utilizan estas para generar un nuevo conjunto de soluciones 
___ 
Pasos:
	1. Definir solución 
	2. Filtrar
		1. Aplicar función objetivo
		2. Ordenar individuos
		3. Seleccionar mejor
	3. Cruzar individuos
	4. Mutación de descendientes
	5. Inserción 
	6. Si se cumple función objetivo => Terminar (si no volver a paso 2)
