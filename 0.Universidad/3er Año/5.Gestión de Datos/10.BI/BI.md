| Nombre       | Definición                                                                                                     |
| ------------ | -------------------------------------------------------------------------------------------------------------- |
| Datos        | Valores concretos ya conocidos                                                                                 |
| Información  | Dato asociado a la realidad (contextualizado)<br>Datos procesados o relacionados con un significado específico |
| Conocimiento | Información acumulada y analizada (comparada, predicción concecuencias, busqueda conexiones)                   |
Datos -> Info:
	- Contextualizando
	- Categorizando
	- Calculando
	- Corrigiendo
	- Condensando (resumiendo datos a forma más concisa)

Info -> Cono:
	- Comparación con otros elementos
	- Predicción de consecuencias
	- Búsqueda de conexiones 
	- Conversación con otros portadores de conocimiento

Definición:
	Transformación: Datos -> Información -> Conocimiento
	Para mejorar proceso de toma de decisiones

# BD Relacional vs Multidemensional
| Caract                            | Relacional                                           | Multidimensional                                                             |
| --------------------------------- | ---------------------------------------------------- | ---------------------------------------------------------------------------- |
| Almacen, acceso y visión de datos | Relacional<br>Tablas<br>SQL                          | Dimensional (Hypercubo,Multicubo)<br>                                        |
| Uso e Incorporación               | OLTP<br>Motor RBDMS<br>Profundiza a nivel detalle    | OLAP<br>Motor multidemensional<br>Profundiza a nivel resumen                 |
| Tamaño y actualización de bd      | GB a TB<br>Indices <br>Consulta y carga en paralelas | GB<br>Actualización durante uso<br>Cabmios pequeños requieren reorganización |
