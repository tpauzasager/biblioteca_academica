Se utiliza un bloque del archivo para contener punteros al resto de los bloques 

| Ventajas                                                                                                        | Desventajas                                                                                                                                                                                                                                                   |
| --------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - Sin problema para agrandar archivo<br>- Sin fragmentación externa<br>- Bueno para acceso secuencial y directo | - Limita el tamaño de los archivos (depende de la canti de punteros almacenados en el puntero)<br>- Archivos pequeños desperdician mucho espacio<br>- Mayor reposicionamiento del cabezal<br>- Mayor espacio ocupado por punteros<br>- Corrupción de punteros |
___
# Modificaciones
Esquema enlazado:
	Se pueden enlazar bloques indices 

Índice multinivel:
	Bloque indice principal apunta a otros bloques índices (y así sucesivamente)

Esquema combinado:
	Usar algunos punteros para bloques de datos y otros para bloques de punteros
