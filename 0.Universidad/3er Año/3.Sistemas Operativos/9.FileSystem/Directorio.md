Definición:
	- Contiene listado de nombres de otros archivos y atributos asociados 
	- **Mapeo** entre nombre de archivo y el archivo en sí
Implementaciones:

| Nombre                  | Ventajas                                                                          |
| ----------------------- | --------------------------------------------------------------------------------- |
| Lista lineal            | - Más fácil de programas<br>- Manipulación compleja<br>- Requiere búsqueda lineal |
| Lista enlazada ordenada | - Más complejo agregar/eliminar entradas                                          |
| Árbol-B                 | - Minimiza tiempo de búsqueda                                                     |
| Tabla hash              | - Lista lineal + estructura de hash                                               |
