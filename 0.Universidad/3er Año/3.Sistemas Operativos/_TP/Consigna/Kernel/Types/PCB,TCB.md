# PCB
- **PID:** Identificador del proceso (entero y único en el sistema)
- **TIDs:** Identificadores de hilos asociados al proceso (único para el proceso y autoincremental)
- **Mutex:** Lista de mutex creados para el proceso durante su ejecución

# TCB
- **TID**: Identificador del hilo
- **Prioridad:** Prioridad del hilo
