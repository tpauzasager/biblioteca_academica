# Detección 
Cada determinado tiempo, se ejecuta el algoritmo de detección de deadlock
Costes: 
- Almacenamiento de info necesaria 
- Ejecución de algoritmo de detección 
- Potencial pérdida por recuperación 
___
# Recuperación 
## Finalizar Procesos
1. Finaliza todos los procesos dentro del deadlock (coste alto)
2. Finaliza de a un proceso hasta solucionar el deadlock (mucho overhead)
## Desalojar Recursos
- Volver proceso a un estado seguro, desde el cual reanudar la ejecución 
- Puede generar inanición (un proceso puede siempre elegirse como víctima)

Factores para selección de proceso víctima:
- Prioridad y tipo 
- Tiempo actual de ejecución 
- Cuantos y tipos de recursos que utiliza y necesita 
- Conviene desalojar a los procesos con más cant de recursos asignados
