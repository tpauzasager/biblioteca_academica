# Definición 
- Conjunto de políticas y mecanismos del SO para gobernar orden de ejecución de los proceso
- Ejecución consiste en alternación entre ráfagas del CPU y ráfagas de E/S
___
# Planificador de Procesos
- Módulo del SO que mueve los procesos entre distintas colas de planificación 
- Limitaciones del proceso
	- Por CPU: Pasa más tiempo procesando que haciendo E/S
	- Por E/S: Al revés
