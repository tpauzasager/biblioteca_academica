# Externa
Definición:
	Ocurre cuando la suma de todos los espacios libres es suficiente para guardar un nuevo proceso, pero estos espacios libres se encuentras dispersos por la memoria
	Es **entre particiones**

Solución: 
	Compactación. Se ponen todos los procesos unos al lado del otro para juntar todos los espacios vacíos 	
____
# Interna
Definición:
	Ocurre cuando el proceso almacenado no ocupa toda la partición que tiene asignada (partición fija) 
	Es **dentro de la partición**

Solución:
	Páginas de menor tamaño
