Definición:
	- Páginas se cargan a memoria física solo cuando se necesitan 
	- Swapping es lazy (solo se mueven pág entre MP y area swap, no todo el proceso)
	- Tiempo de carga de los procesos disminuye 

Bit de cada pág:

| Nombre bit     | Descripción                       |
| -------------- | --------------------------------- |
| Presencia (P)  | 1 -> Pág está en memoria          |
| Modificado (M) | 1 -> Fue modificado recientemente |
Si se hace referencia a una pág con P = 0 => Se lanza interrupción **fallo de página (PF)**

## Page Fault (PF)
Ateción:
	1. Comprobar si dirección es válida para dicho proceso (está dentro de espacio asignado?)
	2. Si es inválida => 
		a. Finalizar proceso
		b. Envíar mensaje de error y devolver flujo de ejecución 
	3. Si es válida => Hay que cargar págs a MP
		- Se lee el disco la pág y se carga en frame elegido
		- Se modifica la tabla de páginas para indicar P=1 
		- Se reinicia la instrucción causante de la interrupción
