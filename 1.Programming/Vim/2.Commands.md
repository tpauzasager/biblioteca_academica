# Start
Type "/" or ":"
# Search in File
| Hotkey    | Description                             |
| --------- | --------------------------------------- |
| /[string] | Finds [string] in file                  |
| [n,N]     | go to [next, previous] result of search |
# History
| Hotkey                            | Description                                                      |
| --------------------------------- | ---------------------------------------------------------------- |
| [/ \| : ] + [up, down]            | scroll though the command history [backwards, foward]            |
| [/ \| : ] + [command] +[up, down] | scroll throught the history of commands that look like [command] |
# Find & Replace
`:%s/<word_to_replace>/<replacement_word>/g

%: In the hole row
g: In the hole file

# Clarifications
- "\\>" & "\\<" : mean the [end, start] of the word  (just one\) 

# Multiple files
| Hotkey                         | Description                  |
| ------------------------------ | ---------------------------- |
| vim [<file_name>]              | Opens all the files with vim |
| :edit <file_name>              | Adds file to buffer          |
| :b {<buffer_num>, <file_name>} | Switch editting file         |
| :ls                            | Shows files from buffer      |
