# Cursor Movement

| Hotkey  | Description                                   | When Uppercased                                    |
| ------- | --------------------------------------------- | -------------------------------------------------- |
| [j,k]   | move cursor[down, up]                         |                                                    |
| [h,l]   | move cursor[left, right]                      |                                                    |
| [w,b]   | move to start of word [foward, backwards]<br> | move by white-space (ignoring non-word characters) |
| [e, ge] | move to end of word [foward, backwards]       | idem                                               |
| [0, $]  | move cursor to [start, end] of line           |                                                    |
| [gg, G] | move to [fist, last] line                     |                                                    |
| [H,M,L] | move to [fist, middle, bottom] line visible   |                                                    |

# Scrolling
| Hotkey                     | Description                                                                                           |
| -------------------------- | ----------------------------------------------------------------------------------------------------- |
| [1, .... , infty] + [gg,G] | place cursor at specifyc line (counting from top)                                                     |
| ctrl [f,b]                 | scroll a whole screen [up, down]                                                                      |
| ctrl [u,d]                 | scroll half screen [up,down]                                                                          |
| ctrl [e,y]                 | scroll one line [up,down]                                                                             |
| zz                         | if cursor at last line of the screen => scroll up to when line with cursor is at middle of the screen |
| z[t,b]                     | moves screen so that line with cursor stays at [top, bottom] of screen                                |
| ctrl [e,y]                 | Move screen one line [up, down] without moving the cursor position                                    |
# Search in Line
[; | ,] to repeat the command [foward, backward]

| Hotkey  | Description                                           | When Uppercased  |
| ------- | ----------------------------------------------------- | ---------------- |
| f+[a,z] | move to first found letter in line                    | Search backwards |
| t+[a,z] | move to one character before the found letter in line | Search backwards |
