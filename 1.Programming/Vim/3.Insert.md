# Start
| Hotkey | Description                                                                     | When uppercased       |
| ------ | ------------------------------------------------------------------------------- | --------------------- |
| i      | start at back of cursor                                                         |                       |
| a      | start at front of cursor                                                        |                       |
| o      | start at new line bellow                                                        | same but over         |
| r      | replace letter in cursor, works for only one letter, then is set in normal mode | same but for hole row |
| s      | deletes de letter in cursor and keeps insert mode                               | deletes hole row      |
